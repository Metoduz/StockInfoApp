# Product Overview

## Stock Info App

A Flutter-based mobile application for tracking and analyzing stock portfolio information. The app displays stock data including current prices, daily changes, and trading hints/analysis for various stocks.

### Key Features

- **Stock Portfolio Display**: Shows a list of stocks with current values, daily changes, and percentage movements
- **Stock Analysis Hints**: Provides trading insights including buy zones, support/resistance levels, and trend analysis
- **Interactive Stock Cards**: Expandable cards showing detailed stock information and hints
- **Multi-platform Support**: Built with Flutter for iOS, Android, Web, Windows, macOS, and Linux

### Target Use Case

Personal stock portfolio tracking with basic technical analysis hints to help users make informed trading decisions. The app focuses on displaying German stocks (DAX companies like BASF, SAP, Mercedes-Benz) with EUR currency support.
